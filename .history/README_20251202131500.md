# gittui
gittui is an open-source command-line tool that brings your Git repository to life inside the terminal. Using ASCII characters and minimalist text graphics, it draws interactive historical charts of branches, merges, and commitsâ€”giving you immediate, intuitive insight into your project's version control history without ever leaving your shell.

## Features

- **Interactive Git History Graph**: Visualize commit history, branches, and merges in a text-based graphical format.
- **Local Repository Support**: Add local Git repositories for visualization and exploration.
- **Repository Cloning**: Clone remote repositories and instantly view their history.
- **New Repository Wizard**: Interactively create new repositories with options for README initialization, .gitignore, and license setup.
- **Clean and Intuitive Interface**: Terminal-based menu navigation for easy usage.

## Installation

Ensure you have Python 3.x installed, then install gittui using pip:

```bash
pip install gittui
```

Or install from source:

```bash
git clone https://gitee.com/with-the-flow/git-tui.git 
cd git-tui
python -m pip install -e .
```

## Usage

Run gittui:

```bash
gittui
```

### Main Operations

- **Add Local Repository**: Select the menu option and enter the local repository path.
- **Clone Remote Repository**: Enter the remote repository URL.
- **Create New Repository**: Follow the interactive process to set repository name, description, path, and initialization options.
- **Exit Program**: Select exit from the menu.
